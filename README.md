# TaskFlow – Professional To-Do List Application

## Overview

TaskFlow is a modern and responsive To-Do List web application developed using HTML, CSS, and JavaScript. The application helps users efficiently manage daily tasks by providing features such as task creation, editing, deletion, priority management, progress tracking, filtering, and local storage persistence.

## Features

* Add, edit, and delete tasks easily.
* Mark tasks as completed or pending.
* Assign task priorities (High, Medium, Low).
* Add categories, due dates, and notes to tasks.
* Search tasks instantly using keywords.
* Filter tasks by:

  * All Tasks
  * Today's Tasks
  * Upcoming Tasks
  * Completed Tasks
  * Priority Levels
* Sort tasks by:

  * Creation Date
  * Priority
  * Due Date
  * Alphabetical Order
* Progress tracking with completion percentage.
* Overdue task detection.
* Responsive design for desktop and mobile devices.
* Data persistence using Local Storage.
* Interactive notifications (Toast Messages).

## How to Run the Project

1. Download or clone the project repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Start managing your tasks.

## Key Functionalities

### Task Management

* Create new tasks with title, category, due date, priority, and notes.
* Update task details through an edit modal.
* Delete unwanted tasks.

### Productivity Tracking

* View total, completed, pending, and overdue tasks.
* Monitor daily progress through a visual progress bar.

### Data Persistence

* All tasks are automatically saved in the browser's Local Storage.
* Tasks remain available even after refreshing or reopening the browser.
